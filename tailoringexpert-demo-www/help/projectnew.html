New Project Wizzard

This is the wizzard to create a new project. Wizzard is performed in four steps:

  1. select catalog version
  2. upload a screeningsheet
  3. check calculated selection vector based on screeningsheet input and update vector if needed
  4. create project

Select catalog version

This is the first step in creating a new project. If only one catalogue is valid at the time of creation, it will be selected. Any notes entered will be added to the initial tailoring created for the project. They are not mandatory.

Upload a screeningsheet

A screening sheet is required as input for project creation and tailoring. Format and content are tenant-specific. Once uploaded, extracted values are displayed.

Check calculated selection vector

A selection vector is calculated based on the given screen sheet. A selection vector defines levels for all parameters of a tailoring. These values are used to select requirements for the initial tailoring. The values can be either modified by

  • selecting a profile
  • adjusting the values by clicking on them

Create project

By clicking the Create button, the project is created with an initial tailoring using the defined selection vector. for the (pre-)selection of requirements.
