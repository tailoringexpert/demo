Project

This is page provides an overview of all tailorings created so far and their releated data

On this page it is possible to

  • edit requirements (pencil icon)
  • compare current tailoring to automated initial one (vector diff icon)
  • open used screeningsheet (account icon)
  • show applied and calculated selection vector (head icon)
  • create tailoring documents (edit icon)
  • management of files/attachments of tailoring (paperclip icon)
  • import requirement selection (Excel icon)
  • viewing add editing notes (message icon)
  • delete tailoring (waste basket icon)

Edit requirements

Opens the page to edit requirements of tailoring. This means that

  • a requirement could be make applicable/not applicable
  • edit requirement text
  • add new requirements

Compare current tailoring to automated initial one

By creating a new tailoring requirements will be made applicable by using the selectionvector. With this function it is is possible to compare the current selected requirements with the initial ones.

open used screeningsheet

Opens dialog on which

  • screeningsheet file can be downloaded
  • shows all parsed parameters of the screeningsheet
  • shows calculated selectionvector based on the screeningsheet

Show applied and calculated selection vector

Opens the dialog to display

  • calculated and
  • applied
selection vector.

Create tailoring documents

Opens starting dialog for creating tailoring documents. It is possible to make signatures applicable and to change names of signees.

Management files/attachments of tailoring

It is possible to add files/attachments to this tailoring. A file/attachment is stored with a MD5 hash. You can

  • upload
  • download
  • delete
    • files.

Import requirement selection

Sometimes there's an existing reference tailoring that's similar to the one a user is working on. The easiest way to transfer the reference tailoring is to create documents in the reference tailoring. In contains an Excel file with all requirements and their current status. This can be imported into the dialogue. This is possible because it is assumed that catalogues are backward compatible. This means that deleted requirements are not deleted in the document, but the text is changed to deleted.

Viewing add editing notes

This icon opens a dialogue box containing all the notes created for the tailoring. This includes the comments made during the catalogue selection step. It is worth mentioning that notes cannot be deleted.

Delete tailoring

Before deleting a tailoring, a confirmation dialogue is displayed. After confirmation, the tailoring is deleted.
